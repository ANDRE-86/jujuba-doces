# Jujuba Doces Experience

Site mobile first para montar orçamento de doces personalizados por etapas.

## Estrutura

- `index.html`: fluxo principal
- `css/style.css`: identidade visual
- `js/app.js`: controle das etapas, carrinho e WhatsApp
- `dados/`: produtos, temas e cores
- `docs/`: regras e roadmap
